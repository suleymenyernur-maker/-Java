# -Java
Банктік жүйе (Java) — анықтама  Банктік жүйе Java тілінде — бұл банк операцияларын (шот ашу, ақша салу, ақша шешу, аудару, баланс көру) автоматтандыру үшін Java бағдарламалау тілінде жазылған ақпараттық жүйе.  Қарапайым тілмен айтқанда, Java банктік жүйесі — клиенттердің шоттарын басқаруға және қаржылық операцияларды орындауға арналған бағдарлама.
